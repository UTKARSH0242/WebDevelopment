Selectors in HTML

Element Selector

The element selector select HTML elements based on the element name

ID Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page , so the id selector is used to select one unique element.

class Selectors

The class selector selects HTML element with a specific class attribute. To select element eith a specific class, write a period(.) character, followed by the class name
